What You Get with Fish MMO

Fish MMO is designed to be flexible, scalable, and efficient. Below are some of the key features you'll get when using this framework.

Login Server

  • Scalable Design, multiple login servers for load balancing
  • Account Authentication
  • Global Character Creation (Characters usable on any World)
  • Character Selection/Deletion
  • World Server Selection

World Server

  • Account Authentication
  • Relay Server for Scene-to-Scene Communication
  • Scene Server Load Balancing

Scene Server

  • Cluster node design, multiple scene servers for a single World Server
  • Account Authentication and Load Balancing
  • Supports up to 100 players per Scene
  • Chat System, Party System, Guild System
  • Character Inventory and Kinematic Character Controller
